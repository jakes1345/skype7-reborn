Phaze

Chat, voice, and video you control — Go relay, encrypted messages, WebRTC media.

Download phazechat.world

Self-host the open-source nexus_server and point native_client / web at your relay. See the repository README.md for build steps.

Modern clients

Fyne desktop & Android from one Go codebase; web client for browsers — same Nexus protocol.

Encryption

Pairwise NaCl box for direct messages; per-member envelopes for groups so the relay never sees plaintext bodies.

WebRTC calls

Voice and video over Pion (native) and browser APIs (web, in progress) with TURN from your relay.

Self-hostable

SQLite-backed relay by default. Your users, your data policy.

Run your own relay

cd nexus_server && go build -o phaze-nexus && ./phaze-nexus

Set PORT, DB_PATH, and TURN env vars as documented in the main README.
