Phaze

Classic Skype-style chat and calls — rebuilt in Go. Open source, self-hostable, no ads.

Download Public relay View on GitHub
MIT license. Public relay: phazechat.world. Pre-1.0 — see the repo README for current capabilities and limits.

Self-host the open-source nexus_server and point native_client / web at your relay. See the repository README.md for build steps.

Classic-style UI

Fyne desktop experience inspired by the Aero-era Skype look.

Encrypted messages

Pairwise NaCl box for direct messages; per-member envelopes for groups so the relay never sees plaintext bodies.

Voice & video

WebRTC voice (PCMU). VP8 video on Linux desktop; Windows + Android use more constrained video paths — see README.

Group chats

Named conversations; per-member ciphertext fan-out so the relay does not see message bodies.

Accounts & security

Email verification, password reset, optional TOTP 2FA, QR sign-in, encrypted local SQLite on desktop, GDPR account-delete.

Self-hostable relay

Run your own Nexus (Docker or go run). Clients can point at your wss:// host. Built-in /metrics and litestream backups when you configure S3 credentials.

Quick start

Download the archive for your platform from GitHub Releases, extract, and run the phaze binary (Windows: phaze.exe). First launch walks you through account creation against the configured server (default: public relay). The desktop app self-updates from GitHub Releases with SHA-256 verification.

Run your own server

docker run -p 8080:8080 -v nexus_data:/data \
  ghcr.io/jakes1345/skype7-reborn-nexus:latest

Or clone the repo and go run ./nexus_server. Point clients at your WebSocket URL (e.g. wss://your-host/ws).

Contribute & beta

Issues and PRs welcome at github.com/jakes1345/skype7-reborn. Use the Bug report and Beta feedback issue templates when testing prereleases.

Not yet the focus of v1: Opus wideband audio, iOS/macOS desktop, multi-relay federation, screen sharing, and full WebRTC parity in the web client — track the README roadmap.
