Phaze

Classic Skype-style chat and calls — rebuilt in Go. Open source, self-hostable, no ads.

Download View on GitHub
MIT license. Public relay: phazechat.world. Pre-1.0 — see the repo README for current capabilities and limits.

Classic-style UI

Fyne desktop experience inspired by the Aero-era Skype look; optional user-scribed assets from an original installer (see repo legal notes).

Voice & video

WebRTC voice (PCMU). VP8 video on Linux/macOS desktop; Android and Windows builds use more constrained video paths — see README.

Group chats

Named conversations; per-member ciphertext fan-out so the relay does not see message bodies.

File transfer

Peer-to-peer over a WebRTC data channel after signaling through Nexus.

Accounts & security

Email verification, password reset, optional TOTP 2FA, QR sign-in, encrypted local SQLite on desktop.

Self-hostable relay

Run your own Nexus (Docker or go run). Clients can point at your wss:// host.

Quick start

Download the archive for your platform from GitHub Releases, extract, and run the phaze binary (Windows: phaze.exe). First launch walks you through account creation against the configured server (default: public relay).

Run your own server

docker run -p 8080:8080 -v nexus_data:/data \
  ghcr.io/jakes1345/skype7-reborn-nexus:latest

Or clone the repo and go run ./nexus_server. Point clients at your WebSocket URL (e.g. wss://your-host/ws).

Contribute & beta

Issues and PRs welcome at github.com/jakes1345/skype7-reborn. Use the Bug report and Beta feedback issue templates when testing prereleases.

Not yet the focus of v1: Opus wideband audio, multi-relay federation, native installers with auto-update, in-browser full client parity, and screen sharing — track the README roadmap.
