Skype 7 Reborn

The Skype you loved, rebuilt from scratch in Go. No ads. No Microsoft. No telemetry.

Download View on GitHub
Free and open source under the MIT license. Public server hosted at phazechat.world. Self-hosting supported.

Classic Skype 7 UI

Pixel-faithful recreation of the 2015 Aero interface with original sounds, emoticons, and Tahoma type.

Voice calls

WebRTC with G.711 audio. One-to-one and group voice works out of the box.

Group chats

Named conversations with multiple members. Typing indicators, read receipts, and offline message delivery.

File transfer

Send any file over a secure WebRTC data channel. No size cap beyond your own bandwidth.

Presence & mood

Online, Away, Do Not Disturb, Invisible. Set a mood message. Broadcast to every friend instantly.

Self-hostable

One Docker container runs the whole relay. Your data on your box. Or use the free public server.

Quick start

Download the archive for your platform, extract, and run skype. First launch registers a new account on the public server.

Run your own server

docker run -p 8080:8080 -v nexus_data:/data \
  ghcr.io/jakes1345/skype7-reborn-nexus:latest

Or clone the repo and go run ./nexus_server. Point clients at wss://your-host/cable.

Contribute

The project is 100% open source. Issues, PRs, and stars welcome at github.com/jakes1345/skype7-reborn. Ideas we haven't built yet: video calls, screen sharing, mobile clients, federated servers.
