Sovereign Verification

Phaze binaries are self-signed and sovereign. Compare your file against checksums.txt on the latest GitHub release (SHA-256 per asset). The list below is illustrative only; always trust the checksum file shipped with the release you downloaded.

Windows SmartScreen / "unknown publisher" warning?
Phaze is code-signed by the project, not by Microsoft's cert-cartel. Windows will say "Windows protected your PC." That's expected. Click More info → Run anyway. Confirm the SHA-256 below matches your download before running. Checksums are how you verify; signatures are how Microsoft charges $200/year for a green badge.
How to verify the hash:
Windows (PowerShell): Get-FileHash Phaze.exe -Algorithm SHA256
Linux / macOS: sha256sum Phaze.linux or shasum -a 256 Phaze.apk
checksums.txt (authoritative) Open the release → download checksums.txt → match the line for your file (e.g. Phaze_windows_amd64.zip, Phaze.apk).
Example commands Windows: Get-FileHash .\Phaze_windows_amd64.zip -Algorithm SHA256 · Linux/macOS: sha256sum Phaze_linux_amd64.tar.gz
