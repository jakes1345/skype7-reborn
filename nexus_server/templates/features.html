All the features of Phaze

📹

Video calling

Peer-to-peer video where supported (VP8 on Linux/macOS desktop today; Windows and Android paths are more constrained — see the README).

📞

Voice calls

Call anyone on Phaze for free. Where the PSTN bridge is enabled on a deployment, verified users may reach phones via the server’s telephony integration — not a consumer mobile app store “minutes” product unless you operate that billing.

💬

Instant messaging

Group and direct messaging with end-to-end ciphertext fan-out on the relay (see README / threat model). Rich emoji and classic emoticon styling depend on assets you provide locally.

📁

File sharing

Send files over a WebRTC data channel after call setup — practical limits depend on network and device memory.

🖥️

Screen sharing

Not shipped in current public builds — on the roadmap after core voice, video, and chat are stable across platforms.

👥

Group chats

Bring people together in named group chats with the same E2EE messaging model as direct chats.
